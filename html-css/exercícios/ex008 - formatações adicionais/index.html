Principais Formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b> (não semântica).

Já nessa, temos um termo em destaque usando a tag <strong> (semântica).

Itálico / Ênfase

Agora um termo em itálico usando a tag <i> (não semântica).

Nessa frase temos um termo em ênfase usando a tag <em> (semântica).

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>.

E no outro parágrafo, temos outro texto marcado no final.

Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado (semântica).

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois (semântica).

Existem também o sublinhado com a tag <u> (não semântica).
